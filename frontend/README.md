# Portfolio Frontend

This is the **frontend** of my personal portfolio built with **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**.  
It’s fully responsive and includes interactive UI features, animations, and particle effects.

---

## Features

- **Interactive Chat:** AI-style messages with typing animation.
- **Animated Carousel:** Infinite horizontal scroll with drag support.
- **Contact Form:** Smooth animations and responsive design.
- **Particles & Animations:** Dynamic background particles and animated section titles.
- **Section Separators:** Gradient and wave transitions between sections.
- **Markdown Loader:** Dynamically render Markdown content for Portfolio Modal.

---

## Tech Stack

- React 18 / Next.js 14  
- TypeScript / Tailwind CSS  
- Framer Motion / Motion/React  
- tsparticles for interactive backgrounds  

---

## Environment Variables

Copy `.env.example` to `.env` and fill in the required values:

```bash
cp .env.example .env
```

Variables include database access, API keys, email settings, etc.  
Never commit the `.env` file containing secrets.

---

## Quick Start

```bash
git clone https://github.com/pauldeveaux/portfolio.git
cd frontend
npm install
npm run dev
```

Open **http://localhost:3000**
